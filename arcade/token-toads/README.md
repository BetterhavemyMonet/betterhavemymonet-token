# Token Toads

A mini Frogger-style arcade game for the Betterhavemymonet-token project.  
Move your token-frog across crypto "traffic" and reach the top to win!

## How to play

- Arrow keys to move.
- Avoid moving blocks.
- Reach the top row to score.

## To run locally

1. Make sure you are in the `/arcade/token-toads/` directory.
2. Run a simple static server (requires Node.js):
    ```
    npx serve .
    ```
   OR, if you have Python installed:
    ```
    python -m http.server
    ```
3. Open your browser to the URL shown in the terminal (often http://localhost:5000 or http://localhost:8000)
4. Play!
